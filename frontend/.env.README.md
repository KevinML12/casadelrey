# Configuración de Variables de Entorno

## Setup Inicial

1. Copia `.env.example` a `.env`:
```bash
cp .env.example .env
```

2. Obtén tu clave pública de Stripe:
   - Ve a: https://dashboard.stripe.com/test/apikeys
   - Copia la clave que comienza con `pk_test_...`
   - Pégala en `.env`:
     ```
     VITE_STRIPE_PUBLISHABLE_KEY=pk_test_tu_clave_aqui
     ```

3. Verifica que el backend esté corriendo en el puerto configurado:
   ```
   VITE_API_BASE_URL=http://localhost:8080
   ```

## Nota de Seguridad

⚠️ **NUNCA** commits el archivo `.env` al repositorio.
Solo commitea `.env.example` como plantilla.
