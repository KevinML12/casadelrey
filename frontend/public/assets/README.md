# Assets - Imágenes del Proyecto

## Hero Banner
Coloca aquí la imagen de fondo del hero banner:
- **Nombre del archivo:** `hero-bg.jpg`
- **Ubicación:** `frontend/public/assets/hero-bg.jpg`
- **Dimensiones recomendadas:** 1920x1080px o superior
- **Formato:** JPG o PNG
- **Contenido sugerido:** Imagen de la iglesia, comunidad joven, o ambiente vibrante

## Instrucciones
1. Descarga o selecciona una imagen de alta calidad
2. Guárdala como `hero-bg.jpg` en este directorio
3. La imagen se mostrará automáticamente en el HeroBanner

## Placeholder Temporal
Mientras no exista la imagen, el componente HeroBanner mostrará un fondo con color sólido.
